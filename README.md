# Business Intelligence Predictive Modeling Application

Customer churn prediction using the Telco Customer Churn dataset with Streamlit, Pandas, and Scikit-Learn.

## How to run

1. Install dependencies (example):
   pip install streamlit pandas scikit-learn seaborn matplotlib

2. Start the app:
   streamlit run app.py

## Dataset

The app loads the dataset from:
assets/datasets/WA*Fn-UseC*-Telco-Customer-Churn.csv
